# Runestone Golang library

Bitcoin Rune protocol golang implement library.

## 前提

已经安装fractalbitcoin全节点

## 安装
 
```go
git clone https://github.com/njskyun/runestone.git
cd runestone/
go mod tidy
```
### 使用
1. cd cmd/runestonecli
2. 修改config.yaml配置文件
3. main.go 文件中有一些基本逻辑，可以自行更改
4. 运行：go run .
